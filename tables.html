Tables

untuk tulisan didalam tag th secara default akan menjadi bold

Tables di html memungkinkan para programmers html untuk mengatur data dalam bentuk kolom dan baris.
Contoh

Person 1 Person 2 Person 3
Emil Tobias Linus
16 14 10

Table tags

  • table : Defines a table
  • th : Defines a header cell in a table
  • tr : Defines a row in a table
  • td : Defines a cell in a table
  • caption : Defines a table caption
  • colgroup : Specifies a group of one or more columns in a table for formatting
  • col : Specifies column properties for each column within a colgroup element
  • thead : Groups the header content in a table
  • tbody : Groups the body content in a table
  • tfoot : Groups the footer content in a table

Sizes

tabel bisa mempunyai ukuran yang berbeda-beda untuk setiap kolom, row atau seluruh tabelnya

Untuk mengubah jarak antar tabel, bisa menggunakan border-spacing property ditabel element, Defaultnya 2px
Saat menggunakan border-collapse sepertinya border-spacing tidak akan bekerja seperti seharusnya

Colgroup

Tapi juga bisa digunakan untuk grouping setiap kolom untuk mempermudah styling.
