HTML style guide

Isinya cuma tips buat nulis code

Source

  • Always declare document type
  • Menggunakan lowercase semua untuk penulisan element
  • Menggunakan lowercase semua untuk penulisan attributes
  • Tutup semua html elements(buat element yang punya tutup
  • selalu gunakan tanda kutip(kutip dua) untuk attributes value
  • Selalu menentukan alt, height dan width untuk img
  • Penulisan lebih baik spaceless.
    Contoh: href=""
  • Hindari Long Code Lines
  • blank lines dan indentation secukupnya
  • Never skip the title elemenet
  • head, html dan body tag bisa tidak ditulis, tapi sangat disarankan untuk tetap menulisnya.
  • Close empty html element.
    Contoh: br yang ada di line ini(buka source codenya).
  • add the lang attribute
  • Use lowercase file names
