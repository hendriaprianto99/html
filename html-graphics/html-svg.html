Apa itu SVG

  • Kepanjangan dari scalable vector graphics
  • digunakan untuk mendefinisikan graphics untuk halaman web
  • w3c recommendation

svg element

  • svg element merupakan sebuah container untuk svg graphics
  • svg element mempunyai beberapa metode untuk menggambar paths, boxex, circles, text, dan graphics image

svg circle

perbedaan antara svg dan canvas

  • svg adalah sebuah bahasa untuk mendeskripsikan 2D graphics di xml
  • canvas menggambar 2D graphics, dengan cepat (with a JavaScript)
  • svg is xml based
  • di svg, setiap pola yang digambar akan diingat sebagai object. jika attributes dari sebuah svg object diubah, maka browser bisa secara otomatis re-render polanya.
  • Canvas di render pixel by pixel. once the graphic is drawn, it is forgotten by the browser. If its position should be changed, the entire scene needs to be redrawn, including any objects that might have been covered by the graphic.
