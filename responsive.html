HTML responsive web design adalah tentang membuat suatu halaman website terlihat bagus disemua device yang membukanya
responsive web design akan otomatis mengatur viewport dan ukuran layar secara otomatis sesuai dengan device user.

Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones)
