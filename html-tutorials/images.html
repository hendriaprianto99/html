Images

Images bisa digunakan dengan menggunakan tag img

attribute yang bisa digunakan di tag img

  • alt : Menunjukkan sebuah alternate text (teks pengganti) yang akan muncul apabila gambar tidak dapat ditampilkan.
  • crossorigin : Mengijinkan website pihak ketiga untuk memperbolehkan akses cross-origin yang dapat digunakan kembali didalam element canvas
  • height
  • width
  • ismap : Menunjukkan bahwa gambar (image) adalah bagian dari gambar peta sisi server (server-side image-map). Jika ditulis, maka koordinat tertentu yang diklik akan dikirim ke server.
  • sizes : Menentukkan daftar ukuran gambar berdasarkan jendela layar browser yang ditampilkan. Melibatkan media query dan ukuran gambar (source-size) yang ingin ditampilkan sesuai kondisi tertentu.
  • src
  • srcset : Memasukkan deretan gambar pilihan yang dapat digunakan oleh browser sesuai kebutuhan atau situasi tertentu (Misalnya, untuk layar kecil, layar dengan resolusi tinggi, dan sebagainya.). Penulisannya, dipisahkan dengan tanda koma (,) per-item gambar.
  • usemap : Mengasosiasikan nama dari peta gambar (image map) yang ditulis dengan diawali tanda pagar (#),
    dan bisa digunakan kalau img berada di dalam attribute a atau button.

HTML Image Tags

  • img : defines an image
  • map : defines image map
  • area : clickable area inside an image map
  • picture : defines a container for multiple image resources

Chapter sum

  • Use the HTML img element to define an image
  • Use the HTML src attribute to define the URL of the image
  • Use the HTML alt attribute to define an alternate text for an image, if it cannot be displayed
  • Use the HTML width and height attributes or the CSS width and height properties to define the size of the image
  • Use the CSS float property to let the image float to the left or to the right

Notes :

  • Format yang didukung : APNG, GIF, ICO, JPEG, SVG, PNG

Image map

clickable areas on an image

The HTML map tag defines an image map. An image map is an image with clickable areas. The areas are defined with one or more area tags.
