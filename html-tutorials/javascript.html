Javascript in HTML

Javascript membuat suatu halaman website lebih dinamis dan interaktif. Digunakanan dengan menggunakan tag script

  • Tag script digunakan untuk mendefinisikan sebuah client-side script(Javascript)
  • script tag contains script statements, atau menggunakan src attribute untuk mengarahkan ke external file
  • Biasanya digunakan untuk memanipulasi gambar, form validation, dan perubaan dinamis konten
  • Untuk mentarget element html menggunakan javasript bisa menggunakan DOM(method)

  • script tag defines a client-side script
  • noscript tag defines an alternate content for users that do not support client-side scripts
