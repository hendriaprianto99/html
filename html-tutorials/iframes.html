Iframes

Iframes digunakan untuk sebuah inline frame. Inline frame digunakan untuk meng-embed dokumen lain kedalam dokumen saat ini

Keterangan

  • iframe secara defaut ada bordernya, pakar border: none untuk menghilangkan border

Iframe untuk target sebuah link

Contoh iframe untuk target sebuah link

Klik disini untuk pindah iframe ke translate

  • Bisa digunakan untuk target sebuah link(jadi link tersebut tidak akan membuka tab baru, melainkan akan terbuka di are iframnya
  • Better pake title untuk memunculkan petunjuk saat hover
  • The HTML iframe tag specifies an inline frame
  • The src attribute defines the URL of the page to embed
  • Always include a title attribute (for screen readers)
  • The height and width attributes specifies the size of the iframe
  • Use border:none; to remove the border around the iframe
