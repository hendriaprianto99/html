From ASCII to UTF-8

ASCII adalah karakter standard pengkodean pertama, ASCII menentukan 128 karakter yang digunakan di Internet. (0-9, a-z, dan beberapa karakter)

  • ISO-8859-1 adalah default character set untuk HTML 4. Character ini men-supporte 256 character codes yang berbeda. HTML 4 juga men-support UTF-8.
  • ANSI (Windows-1252) adalah karakter set original windows. Sama dengan ISO-8859-1, hanya saja punya 32 lebih karakter daripada ISO-8859-1
  • HTML 5 spesifikasi mendorong para programmer untuk menggunakan UTF-8 charset, yang mana UTF-8 mendukung hampir semua karakter dan simbol yang ada di dunia

The ASCII Character Set

  • ASCII uses the values from 0 to 31 (and 127) for control characters.
  • ASCII uses the values from 32 to 126 for letters, digits, and symbols.
  • ASCII does not use the values from 128 to 255.

The ISO-8859-1 Character Set

  • ISO-8859-1 is identical to ASCII for the values from 0 to 127.
  • ISO-8859-1 does not use the values from 128 to 159.
  • ISO-8859-1 is identical to UTF-8 for the values from 160 to 255.

The UTF-8 Character Set

  • UTF-8 is identical to ASCII for the values from 0 to 127.
  • UTF-8 does not use the values from 128 to 159.
  • UTF-8 is identical to both ANSI and 8859-1 for the values from 160 to 255.
  • UTF-8 continues from the value 256 with more than 10 000 different characters.
