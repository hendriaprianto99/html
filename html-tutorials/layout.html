Html layout techniques

Ada 4 teknik untuk membuat multikolom layout

  • CSS framework = penggunaanya tergantung ketentuan dari frameworknya sendiri
  • CSS float property = saat sudah menggunakan float, inget harus di clear floatnya (Lengkapnya nanti di materi CSS)
  • CSS flexbox = penggunannya lebih memastikan sebuah halaman web saat dibuka di ukuran layar yang berbeda(Lengkapnya nanti di materi CSS)
  • CSS grid = pakai kolom dan row, tanpa harus menggunakan float dan positioning(Lengkapnya nanti di materi CSS
