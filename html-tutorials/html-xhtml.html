Apa itu XHTML

  • XHTML adalah singkatan dari EXtensible HyperText Markup Language
  • XHTML lebih ketat, seperti XML-based of HTML
  • XHTML adalah HTML yang didefinisikan sebagai aplikasi XML
  • XHTML disupport oleh majornya web browser

Kenapa XHTML

  • XML adalah sebuah markup language dimana semua documents harus ditandai dengan benar (be "well-formed").
  • XML dibuat untuk membuat HTML lebih extensible dan flexible untuk bekerja dengan format data lainnya(seperti XML). Selain itu, biasanya HTML akan mengabaikan errors yang ada didalam halaman web.
    Jadi, XHTML hadir dengan stricter error handling.

Perbedaan yang penting antara XHTML dan HTML

  • wajib
  • xmlns attribute di html wajib
  • html, head, title, dan body wajib
  • Elements harus selalu properly nested
  • Elements must always be closed
  • Elements must always be in lowercase
  • Attribute names must always be in lowercase
  • Attribute values must always be quoted
  • Attribute minimization is forbidden

XHTML - <!DOCTYPE ....> Is Mandatory

  • An XHTML document must have an XHTML declaration.
  • The html, head, title, and body elements must also be present, and the xmlns attribute in html must specify the xml namespace for the document.
