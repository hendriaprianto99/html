Cascading Style Sheet

Cascading Style Sheets (CSS) is used to format the layout of a webpage.

CSS digunakan untuk membarikan style untuk sebuah webpage

CSS bisa memberikan kita akses untuk mengatur warna, font, ukuran teks, spacing, positions dan banyak lagi.
Singkatnya, CSS mengizinkan kita untuk menata sebuah webpage sesuka kita.
