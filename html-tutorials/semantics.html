HTML Semantics

Semantic elements = elements with a meaning.

Source


Apa itu semantics element?

Elemen semantik dengan jelas menjelaskan artinya bagi browser dan developer.
- span dan div adalah element non-semantics
- table, form, dan article element adalah semantic element

Section element

Menentukan element di sebuah halaman web.
Menurut w3c HTML documentation :"A section is a thematic grouping of content, typically with a heading."

Section element bisa digunakan untuk chapters, introduction, news item dan contact information.
A web page could normally be split into sections for introduction, content, and contact information.

article element

the article element specifies independent, self-contained content.
article element bisa digunakan untuk:

  • Forum posts
  • Blog posts
  • User Comments
  • Product cards
  • Newspaper articles

    nav element

    The nav element defines a set of navigation links.

    aside

    the aside element defines some content aside from the content it is placed in (like a sidebar).

    konten aside harus terkait dengan konten sekitarnya

    figure & figcaption

    • figure tag menentukan self-contained content
    • figcaption menentukan caption untuk tag figure. Bisa digunakan sebagai first atau last-child dari tag figure

    Why semantic elements?

    According to the W3C: "A semantic Web allows data to be shared and reused across applications, enterprises, and communities."
